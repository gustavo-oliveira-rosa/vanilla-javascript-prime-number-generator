O que são números primos?

Números primos são números naturais maiores que 1 que possuem apenas dois divisores: 1 e ele mesmo.

Exemplos: 2, 3, 5, 7, 11, 13, 17, 19, 23, ...

Este gerador usa o algoritmo Crivo de Eratóstenes para encontrar todos os números primos até o limite especificado de forma eficiente.
